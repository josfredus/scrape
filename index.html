Type subreddit names in the order you want to see them appear, and press Enter.
Follow subreddits with a sorting config if you want to specify one, just like so: aww (top, week)
Enter a " / " slash between spaces to start a new list of subreddits, that will play out in parallel with the other lists.
You can also add the following arguments to your command:
--shuffle, -s
Shuffle the order of the subreddits you typed in.
--no-shuffle, -ns
(By default) Preserve the order of the subreddits you typed in.
--skip, -sk, --skip N, -sk N
Auto-skip to the next content every N seconds (default: 10s).
--no-skip, -nsk
(By default) Don't automatically skip to the next content.
--reverse, -r, --reverse N, -r N
Start at round N (default: 5th round), auto-skip will go towards 1st round if enabled.
--no-reverse, -nr
(By default) Don't do what's described just above.
--hot, -h, --new, -n, --rising, -rs
--top P, -t P, --controversial P, --contro P, -c P
Use a sorting config by default for all subreddits you haven't given a sorting config to. P can be any of the following periods: hour, day, week, month, year, all.
Once you've entered your command, use the following controls during the exhibition:
Escape, Backspace
Go back to the command screen.
Arrow keys: Down, Right (downstream) / Up, Left (upstream)
Skip to another content in the desired direction.
Space
Toggle auto-skip.
Enter
Show titles of content / Hide every description / Bring back captions.
Shift
Enter navigation mode between viewports: use left and right arrows to cycle between viewports, use down and up arrows to skip content for the currently selected viewport only.
Control
Lock / Unlock the currently selected viewport; or Unlock everything if not in navigation mode.
